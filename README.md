### Cola에 대해

> Cola는 이 테마에 제가 붙인 이름입니다. Hexo 블로그 엔진을 기반으로 한 테마로, 깔끔하고 세련되며 간결한 스타일을 가지고 있습니다.

### 할 일 목록/로그

- [x] 프로젝트 초기화
- [x] 레이아웃 완성
- [x] 홈페이지 구축
- [x] markdown과 페이지 변환 구현
- [x] 로그 페이지 구축
- [x] 친구 링크 페이지 구축
- [x] 분류 페이지 구축
- [x] 태그 페이지 구축
- [x] 도구 페이지 구축
- [x] 모바일 기본 지원
- [x] pjax 추가로 페이지 전환 시 음악이 멈추지 않도록 함
- [x] 물결 효과 도입
- [x] hexo 테마에 pr 제출, 통과되기를 희망
- [x] jQuery의 cdn이 너무 느려서 로컬로 변경
- [ ] 다크 테마 추가 검토
- [ ] GSAP 도입하여 인터랙션 최적화 검토
